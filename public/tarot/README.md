# 타로 카드 이미지

`components/TarotCard.tsx`는 각 카드의 `id`를 기준으로 이 폴더에서 이미지를 찾습니다:

```
public/tarot/{card.id}.png
```

예: `major-0.png` (The Fool), `wands-1.png` (Ace of Wands) ...
전체 78개 파일명 목록은 `FILENAMES.txt` 참고.

파일이 없는 카드는 자동으로 이모지 엠블럼으로 대체되므로, 이미지를 한꺼번에 다 준비하지 않고
일부만 먼저 넣어도 앱이 깨지지 않습니다.

## 라이선스 참고

1909년 초판 라이더-웨이트-스미스 덱(파멜라 콜먼 스미스 원화)은 미국 등에서 퍼블릭 도메인입니다.
다만 온라인에서 쉽게 구할 수 있는 스캔본 중 상당수는 이후 재발행·복원·컬러 보정을 거친
상업판(예: U.S. Games Systems 발행본)이라 별도 저작권이 걸려 있을 수 있습니다.
실제로 사용하시는 이미지의 출처와 라이선스를 직접 확인해 주세요.
